Two GitHub Actions workflows run on push to master: (1) CI (.github/workflows/ci.yml) — Python 3.13, uv sync --frozen, ruff check, pyright type checking, pytest, uv build. (2) Publish (.github/workflows/publish.yml) — builds wheel+sdist, uploads artifacts, then publishes to PyPI using Trusted Publisher OIDC (no API tokens). The publish job uses pypa/gh-action-pypi-publish with skip-existing:true. Local quality gate mirrors CI: ruff check, pyright, pytest, uv build. Version is defined in src/cwmem/__init__.py and read by hatch via [tool.hatch.version] path config.
