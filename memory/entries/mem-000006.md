High-risk operations (sync export/import) support a reviewable plan workflow: (1) 'cwmem plan sync-export' generates a PlanArtifact JSON capturing workflow, request_hash, options, summary, and impacted resources; (2) Human or agent reviews the plan; (3) 'cwmem validate --plan FILE' checks the plan against current state via request_hash comparison; (4) 'cwmem apply --plan FILE' executes with drift protection. Additionally, --dry-run copies the database to a temp directory and runs mutations on the copy, returning previews without side effects. --idempotency-key enables safe retries by recording (key, command_id) -> response pairs.
